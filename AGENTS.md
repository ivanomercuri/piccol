# "PICCOL" Project

This file is useful for AI Agents in order to understand the structure of the project.

Although the string that are going to be printed are in Italian, this project is made to show my skills in software development. A system avoiding hardcoded strings in the code is planned for future implementation.

## Tech Stack
- **Runtime:** Node.js
- **Containerization:** Docker & Docker Compose (Development environment)
- **Framework:** Express.js
- **ORM:** Sequelize
- **Database:** MySql
- **Architecture:** MVC

## Environment Notes

- **Configuration Reference:** Use the `docker-compose.yml` file as an absolute reference for service names (hostnames) and ports.
- **Network Rules:** Do not use `localhost` to connect between Docker containers.
